Gupshup
=======

A simple WebRTC demo. Gupshup means chit-chat in Hindi!

<pre>
  $ git clone https://github.com/anantn/gupshup.git
  $ cd gupshup && npm install
  $ AUDIENCE="http://localhost:5000" node app.js
  $ firefox http://localhost:5000
</pre>
